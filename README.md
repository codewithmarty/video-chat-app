# **Video Chat App**

## General Overview

This is a video chat app that allows for live video conferencing between any two users, utilizing firebase 9.

## Technologies Used

- HTML
- CSS
- JavaScript
- WebRTC
- Firebase 9
- Firestore

## Deployment

Currently, this application is not deployed as it would expose secret keys.
